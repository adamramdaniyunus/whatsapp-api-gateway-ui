# WhatsApp Gateway & Bot Simulator

This project is a WhatsApp Gateway and Bot Simulator built with React and Vite.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
